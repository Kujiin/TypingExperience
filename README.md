# TypeFlow Studio

A polished typing trainer built with HTML, Tailwind CSS, and vanilla JavaScript.

## Check it out!

https://kujiin.github.io/TypingExperience/

## Run locally

Open index.html in a browser, or serve the folder with a simple static server.

Example:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000.
